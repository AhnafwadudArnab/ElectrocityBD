================================================================================
                        TOKEN কোথায় পাবেন?
================================================================================

🔹 DEVELOPMENT MODE (এখন):
   - Email দেওয়ার পর screen এ একটা orange box দেখাবে
   - সেখানে পুরো token দেখা যাবে
   - Copy button (📋) এ ক্লিক করে copy করতে পারবেন
   - অথবা mouse দিয়ে select করে Ctrl+C করতে পারবেন

🔹 PRODUCTION MODE (পরে):
   - Token আপনার email এ পাঠানো হবে
   - Email check করে token copy করবেন
   - তারপর same process


================================================================================
                        TOKEN এর বৈশিষ্ট্য
================================================================================

✅ Token Length: 64 characters (খুব লম্বা)
✅ Token Validity: 1 ঘন্টা (তারপর expire হয়ে যাবে)
✅ One-time Use: শুধু একবার use করা যাবে
✅ Secure: Random generated, guess করা impossible

================================================================================
                        TROUBLESHOOTING
================================================================================

❌ PROBLEM: Token দেখাচ্ছে না
✅ SOLUTION: 
   - Email টা সঠিক দিয়েছেন কিনা check করুন
   - Database এ email টা আছে কিনা verify করুন
   - Backend running আছে কিনা check করুন

❌ PROBLEM: Token copy হচ্ছে না
✅ SOLUTION:
   - Copy button (📋) এ ক্লিক করুন
   - অথবা mouse দিয়ে select করে Ctrl+C করুন
   - "Token copied to clipboard!" message দেখাবে

❌ PROBLEM: "Invalid or expired token" error
✅ SOLUTION:
   - Token টা 1 ঘন্টার বেশি হয়ে গেছে (expire)
   - আবার নতুন token request করুন
   - অথবা token টা ভুল paste করেছেন

❌ PROBLEM: Password reset হচ্ছে না
✅ SOLUTION:
   - Password minimum 6 characters দিয়েছেন কিনা check করুন
   - Confirm password match করছে কিনা verify করুন
   - Token সঠিক আছে কিনা check করুন


================================================================================
                        IMPORTANT NOTES
================================================================================

⚠️ Development vs Production:
   - Development: Token screen এ দেখাবে (testing এর জন্য)
   - Production: Token email এ যাবে (security এর জন্য)

⚠️ Token Security:
   - Token কাউকে share করবেন না
   - Token দিয়ে যে কেউ আপনার password change করতে পারবে
   - 1 ঘন্টার মধ্যে use করুন

⚠️ Password Requirements:
   - Minimum 6 characters
   - কোন special character লাগবে না (optional)
   - Strong password recommended
